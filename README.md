## ğŸ“Œ Brief Description

Expense Splitter is a simple frontend web application that helps groups of people split expenses in a fair and transparent way. It allows users to add members, record shared expenses, and automatically calculate how much each person owes or is owed.

All data is stored using browser localStorage, so the information remains available even after refreshing the page. This project does not use any backend, APIs, or authentication.

- Clear settlement summary  
- Data persistence using localStorage
